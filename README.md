# Drought Risk Prediction Web App

This is a Streamlit web application for drought risk prediction using trained machine learning models.

## How to Run the Project

### 1. Open the Project Folder

Open the project folder in Visual Studio Code:

```text
drought_risk_web_app
```

---

### 2. Create a Virtual Environment

Run this command in the VS Code terminal:

```bash
py -3.11 -m venv .venv
```

---

### 3. Activate the Virtual Environment

Run:

```bash
.venv\Scripts\activate
```

After activation, you should see:

```text
(.venv)
```

at the beginning of the terminal line.

---

### 4. Install Required Libraries

Run:

```bash
pip install -r requirements.txt
```

---

### 5. Run the Web Application

Run:

```bash
streamlit run app.py
```

The web app will open automatically in the browser.

---

## Important Note

This project requires:

```text
Python 3.11
scikit-learn==1.6.1
```

The correct version is already included in `requirements.txt`.
